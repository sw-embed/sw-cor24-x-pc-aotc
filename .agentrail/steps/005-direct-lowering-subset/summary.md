Implemented direct p-code to COR24 assembly lowering in cor24-emit-asm crate. Supports core opcode subset: stack ops (push/dup/drop/swap/over), arithmetic (add/sub/mul/neg), all 6 comparisons (eq/ne/lt/le/gt/ge), control flow (jmp/jz/jnz), frame management (enter/leave), and variable access (loadl/storel/loadg/storeg). Unsupported opcodes emit TODO comments. Implemented pcode-aotc CLI that reads .p24 files and writes .s assembly output. Memory-based eval stack using hardware push/pop, absolute jumps via la+jmp, comparison inversion pattern for ne/le/ge. 14 unit tests, all 6 test p24 files emit successfully. No TOS caching yet (deferred to step 011).
